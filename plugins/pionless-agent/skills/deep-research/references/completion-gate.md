# Completion Gate

The task is complete only when **all** of the following are true.

- The main question is answered directly.
- Major claims are backed by evidence — at minimum independently confirmed when load-bearing (the verifier agent owns the independence and tier rules).
- Important contradictions were checked and surfaced in the report.
- Uncertainty is called out explicitly (no smoothing).
- The report is organized for decision-making, not as a raw dump.

## If blocked

If the loop terminates before the gate passes, the report must end with:

- **What you verified** — the firm conclusions reached.
- **What remains uncertain** — gaps and contested claims.
- **The minimum next action required** — what the user or a follow-up job would need to do to close the gap.

A blocked-but-honest report is more useful than a smooth-but-overreaching one.
